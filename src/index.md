# Welcome to your new site

This is a starter template using [Eleventy](https://www.11ty.dev/),
[Vite](https://vitejs.dev/), and [Tailwind CSS](https://tailwindcss.com/).

## Features

- **Eleventy v3** for static site generation
- **Vite** for fast development and optimised builds
- **Tailwind CSS v4** for utility-first styling
- **Vitest** for testing
- **ESLint and Stylelint** for code quality
- **RSS feed** for blog posts
- **Accessible navigation** with mobile menu and skip links

## Getting started

Edit the files in `src/` to customise your site:

- `src/index.md` --- this homepage
- `src/about.md` --- about page
- `src/_includes/base.njk` --- base layout template
- `src/assets/main.css` --- styles and colour scheme
- `src/_data/` --- global data files

## Next steps

1. Update the site metadata in `eleventy.config.js`
2. Customise the colour scheme in `src/assets/main.css`
3. Add your own content pages
4. Create blog posts in `src/posts/`
