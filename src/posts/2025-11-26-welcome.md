# Welcome to the blog

This is a placeholder post to demonstrate the blog functionality. Replace or
delete this post and add your own content.

## Getting started with posts

To create a new post, add a markdown file to `src/posts/` with frontmatter like
this:

```yaml
---
title: Your post title
description: A brief description for SEO and feeds.
date: 2024-01-15
---
```

Posts are automatically sorted by date (newest first) and included in the RSS
feed.

## Features

- Automatic date formatting
- RSS feed generation
- Post listing page at `/posts/`
- Markdown with smart typography --- including em dashes
